Computes the mean value of the maximum membership function of a fuzzy set.

The mean value is computed while integrating over the fuzzy set. The integration algorithm is the midpoint rectangle method (https://en.wikipedia.org/wiki/Rectangle_method).

Parameters
termis the fuzzy set
minimumis the minimum value of the fuzzy set
maximumis the maximum value of the fuzzy set
Returns
the mean \(x\)-coordinate of the maximum membership function value in the fuzzy set
